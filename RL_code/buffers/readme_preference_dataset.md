### 🍰 功能介紹：

* `add(image_a, image_b, preference)`：加入一筆 VLM 回傳的偏好資料，像：「我覺得圖 A 比較整齊」
* `sample_batch(batch_size)`：可以給 reward model 訓練時取出一批偏好資料
* `meta` 可以加一些像 step/episode ID、timestamp 或 prompt 等等都行喔♡
